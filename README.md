# 🎨 Image Colorization Project

This project focuses on automatic colorization of grayscale images using two approaches:

* Traditional image processing
* Deep learning (OpenCV DNN)

---

## 📂 Project Structure

```
image-colorization/
├── images/
├── results/
├── results-dl/
├── comparison/
├── models/
├── assets/
├── main.py
└── requirements.txt
```

---

## 🚀 How to Run

```
pip install -r requirements.txt
python main.py
```

---

## 📊 Results

The deep learning approach produces more realistic and visually appealing results compared to traditional methods.

---

⚠️ Model Files

Large model files are not included in this repository.

Download the required model from:
https://github.com/richzhang/colorization/releases

After downloading, place the .caffemodel file inside the models/ folder.
